# 개요
빌드 과정은 추가적인 도구와 점검을 포함하기에 좋은 위치다. 중요한 정보 중 하나는 코드 커버리지다.

자바 생태계에서 코드 커버리지를 확인하는 주요 방법중 하나는 JaCoCo다. 

이를 활성화하기 위해서는 pom.xml 에 plugin 섹션에 플러그인을 추가하면 된다.
